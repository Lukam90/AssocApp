## Maquettage du projet de site "La Savoie ou Bien ?" (5)

**Tâches, opérations, conditions**

Le premier projet de formation consistait à créer un site vitrine basé sur une maquette.

J'ai choisi le thème des séjours en montagne et de la Savoie.

La première version s'est basée sur une interface de bureau.

**Moyens**

J'ai réalisé la maquette avec LucidChart.

J'ai réalisé le design avec HTML et CSS, en m'inspirant d'un modèle de site de voyages d'un tutoriel sur Youtube.

J'ai utilisé SCSS comme préprocesseur CSS.

J'ai utilisé Python et Jinja pour la compilation de fichiers HTML.

J'ai utilisé les media queries pour le responsive et l'adaptation du design à différentes tailles d'écran.

J'ai utilisé DevTools pour tester le design responsive.

**Travail seul / équipe**

J'ai travaillé seul et en autonomie.

## Développement des composants métiers du projet SymCRM (19)

**Tâches, opérations, conditions**

Le projet SymCRM est basé sur le framework Symfony, lui-même basé sur le modèle MVC (Modèle Vue Contrôleur).

La couche Modèle concerne les classes liées aux tables de la base de données.

La couche Vue concerne les fichiers liés aux pages visibles par un utilisateur.

La couche Contrôleur gère les intéractions avec les classes modèles et les vues.

**Moyens**

J'ai utilisé le framework Symfony basé sur PHP.

J'ai utilisé l'ORM Doctrine pour la définition des modèles.

J'ai utilisé le moteur de templates Twig, intégré à Symfony par défaut.

**Travail seul / équipe**
